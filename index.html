Here's why Javascript is Superior


Reason 1: No Compiling


At least... not yet. Maybe for bigger projects, but so far Javascript is much faster to use / test / etc cause you don't have to constantly build the whole project and launch it

Reason 2: No more convoluted classes


Wanna make a giraffe in JS? Make an object with weight, height, age, etc. Boom! Easy giraffe. Wanna make a giraffe in C++? Have fun with public and private variables, functions, etc. And the functions can only really be used by the giraffe. And you can't add the giraffes together with out reason three.

Reason 3: No Operator Overloading


Can't add giraffes with C++? Of course not. C++ doesn't know how to add giraffes. You need to make dumb operator overloading functions so that the '+' symbol can learn how to do it. With JS, just make a nice little function that adds the giraffe's properties together. Much more intuitive. Hallelujah.
